# SNAKE
## A retro classic written in C and SDL.

<hr />

### Run the game
#### Mac
<code>brew install sdl2 sdl2_mixer</code>

#### Linux
<code>sudo apt install libsdl2-dev libsdl2-mixer-dev</code>

#### Compile
<code>
clone https://github.com/jamescarr0/snake<br>
cd snake <br>
cmake -Bbuild/<br>
cd build<br>
make<br>
</code>

##### Binary
./snake<br>

<hr />

#### Play SNAKE!
<code>./snake</code>
